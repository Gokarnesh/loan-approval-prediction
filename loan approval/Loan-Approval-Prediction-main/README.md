# Loan Approval Prediction System

## Overview
This project predicts whether a loan application will be approved or rejected using Machine Learning.

The prediction is based on:
- Applicant Income
- Loan Amount
- Credit History
- Education

---

# Objective
- Predict loan approval status
- Reduce manual processing errors
- Speed up loan approval decisions
- Help banks assess risk efficiently

---

# Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

---

# Dataset Features

| Feature | Description |
|---|---|
| ApplicantIncome | Applicant monthly income |
| LoanAmount | Loan amount |
| Credit_History | Credit score/history |
| Education | Graduate or Not Graduate |
| Loan_Status | Approved or Rejected |
